ファイルの命名規則 / File naming rule
ファイル名は、Year_JournalName or BookName or ConferenceName_InitialOfAuthorFirstName-AuthorLastName.pdfのようにしてください。
つまり、以下の命名規則に則っていないファイル名は無効としています。
[4桁の半角数字]_[1文字以上の半角英数字]_[1文字の半角英字]-[1文字以上の半角英字(ウムラウト等の特殊アルファベット文字も可)].[1文字以上の半角英字]
学士論文、修士論文、博士論文の場合は，JournalName or...の部分をそれぞれ，Bachelor, Master, PhDに置き換えてください．
([]はファイル名に含めないでください。説明用の表現です。)

Please rename your file name like this: Year_JournalName or BookName or ConferenceName_InitialOfAuthorFirstName-AuthorLastName.pdf
Please follow the following naming rule.
[4 digit number]_[One or more alphabetic characters]_[One alphabetic character]-[One or more alphabetic characters (Supporting special characters including Umlaut)].[One or more alphabetic characters]
(Do not include [] in the file name. It is an expression for explanation.)
