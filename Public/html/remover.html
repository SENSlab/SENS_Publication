使い方 / How to Use
  1. カテゴリーと年を選択して検索ボタンを押す。 / Select a category and a year and push a search button.
  2. 削除したい行のマイナスボタンを押す。 / Push a minus button which you want to delete.
    マイナスボタンは一つしか選択できません。 / You can select only one minus button.
  3. 削除ボタンを押す。 / Push a delete button.
