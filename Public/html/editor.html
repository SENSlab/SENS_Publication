使い方 / How to Use
  1. カテゴリーと年を選択して検索ボタンを押す。 / Select a category and a year and push a search button.
  2. 編集したい行の鉛筆ボタンを押す。 / Push a pencil button which you want to edit.
    鉛筆ボタンは一つしか選択できません。 / You can select only one pencil button.
  3. 編集する項目を選択する。 / Select a element which you want to edit.
  4. データを編集する。 / Edit a data.
  5. 削除ボタンを押す。 / Push a delete button.
注意点 / Notes
  • ローディングバーが出ている間は操作しないでください。 / Don't manipulate PC when showing loading bar.
  • 画面が固まってもしばらく待ってみてください。 / Please wait if the display looks freeze.
